**Plugin for Gothic 2 Night of Raven**

This plugin creates a BVH tree for a specific visual object with .3DS model
It improves average FPS in 10%-50% range (depens on vobs amount in the scene)


**Usage requirements**

Union 1.0l or newer


**Build requirements**

Visual Studio 2015-2019
You can use the newer version, but you'll have to retarget the .NET Framework version to newer one (4.x).
This is because 2019 release is the last one to support .NET Framework 4.0.
